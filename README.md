<h1 align="center">Kermes</h1>

<p align="center">
  Kermes é um aplicativo simples e rápido para leitura de mangás, integrado com a API do <b>MangaDex</b>.  
  O objetivo é oferecer uma experiência leve, prática e organizada para quem gosta de ler mangás diretamente do celular.
</p>


---

## Funcionalidades

### Catálogo de obras
Explore títulos disponíveis no MangaDex, com capas organizadas em tamanho padronizado.  
Pesquise por mangás e encontre facilmente o que deseja ler.

## Tecnologias

- Flutter  
- Provider (gerência de estado)  
- Integração com MangaDex API  

---

## Instalação

1. Baixe o arquivo **`Kermes_0.0.3.apk`** disponível abaixo.  
2. Instale em seu dispositivo Android.  
   > Pode ser necessário ativar a opção *Instalar de fontes desconhecidas* nas configurações do dispositivo.
